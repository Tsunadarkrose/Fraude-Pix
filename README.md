# Vero — Landing Page Fraude Pix

Landing page responsiva para um escritório de advocacia que orienta pessoas em casos de fraude bancária e golpe via Pix.

## Publicar na Vercel

1. Importe este repositório na Vercel.
2. Mantenha o framework detectado como **Next.js**.
3. Não é necessário configurar variáveis de ambiente.
4. Clique em **Deploy**.

## Antes da publicação definitiva

Substitua os placeholders presentes em `app/page.tsx`:

- `WHATSAPP_NUMBER`;
- `PHONE_DISPLAY`;
- nome do escritório;
- nome e foto do advogado;
- número da OAB;
- e-mail e localização;
- política real de análise inicial.

## Desenvolvimento local

```bash
npm install
npm run dev
```
